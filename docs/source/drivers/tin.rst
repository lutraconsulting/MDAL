================================================================================
Esri TIN -- Elevation model as TIN
================================================================================

.. shortname:: Esri TIN

.. built_in_by_default::

MDAL supports reading of Esri TIN_ format. This format is used to store elevation in triangulated irregular network (TIN).

.. _TIN: https://en.wikipedia.org/wiki/Esri_TIN
