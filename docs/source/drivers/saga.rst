SAGA Flow -- Raster flow directions from SAGA GIS
================================================================================

.. shortname:: SAGA Flow

.. built_in_by_default::

MDAL supports reading of the `SAGA Flow`_ files. The file needs to be further processed by the Crayfish_ plugin.
