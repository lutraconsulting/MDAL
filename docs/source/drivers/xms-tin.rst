================================================================================
XMS TIN -- ASCII TIN
================================================================================

.. shortname:: XMS TIN

.. built_in_by_default::

MDAL supports reading of `XMS TIN`_ format. This format is used to store triangulated irregular network (TIN) in ASCII format.

.. _XMS TIN: https://www.xmswiki.com/wiki/TIN_Files
